    Example:

## Exercise 2

    Example:
        Input: [3, 1, 2]
        Output: Total Swaps = 2

    
## Exercise 3

Trace Bubble Sort. Show the list after each full pass of Bubble Sort.

   
    Example:
        Input: [6, 3, 5, 2]
        Output: 
                Pass 1: [3, 5, 2, 6]
                Pass 2: [3, 2, 5, 6]
                Pass 3: [2, 3, 5, 6]
